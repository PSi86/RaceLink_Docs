# Screenshots

This directory holds PNG screenshots of the RaceLink WebUI used by
the documentation pages.

> The PNGs are **generated**, not hand-edited. To (re)generate, see
> [`scripts/generate_screenshots.py`](../../../scripts/generate_screenshots.py)
> and the script's [README](../../../scripts/README.md).

If you opened a docs page and saw a broken image link with a
filename ending in `.png`, the corresponding capture has not been
run yet. Run the script — it walks every page registered in its
`CAPTURES` list and writes the PNGs into this folder.
